# Equation-to-code map for Chau et al. 2025

Scope inspected:

- Main repository docs: `README.md`, `paper/README.md`, `pyproject.toml`.
- Reproduction and figure scripts under `paper/`.
- Core model code under `src/niarb/`, especially `nn/modules/v1.py`, `nn/modules/kernels.py`, `special/resolvent.py`, `weights.py`, `numerics.py`, `integrate.py`, `zero_crossing.py`.
- Fitting configs under `paper/model_fits/no_disorder/` and `paper/model_fits/disordered/`.
- Main paper PDF and supplement PDF were inspected visually from rendered pages because text extraction tools were unavailable in this workspace.

Notation caveat: code indices use postsynaptic first, presynaptic second. Thus paper `W_{\alpha\beta}` maps to code matrix entries `W[post_cell_type, pre_cell_type]`, with excitatory/inhibitory signs applied by presynaptic `sign`.

## Key files to read next when changing weight functions

If the goal is to define new weight functions between two neurons, start here:

1. `src/niarb/nn/modules/kernels.py`: defines pairwise kernel building blocks such as `Laplace`, `Tuning`, `SpaceGain`, `SpaceOriGain`, `RankOne`, and `Monotonic`.
2. `src/niarb/nn/modules/v1.py`: composes the full connectivity kernel in `V1.__init__`, especially lines 691-769.
3. `src/niarb/weights.py`: converts a continuous pairwise kernel into finite matrix weights.
4. `paper/model_fits/*/*.toml`: selects kernels and parameters for reproduction experiments.
